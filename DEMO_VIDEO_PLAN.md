## PRODUCTION NOTES

### **Recording Setup:**
1. **Screen Recording:**
   - Use 1080p or 4K resolution
   - Record at 60fps for smooth playback
   - Clean desktop (close unnecessary apps)
   - Use a clean browser window

2. **Product Demo Preparation:**
   - Have sample files ready (PDFs, Excel, videos)
   - Pre-populate a demo project with good data
   - Ensure all features work smoothly
   - Use real data that looks impressive

3. **Visual Enhancements:**
   - Add subtle zoom-ins on important UI elements
   - Use smooth transitions between sections
   - Consider adding subtle cursor highlights
   - Keep UI clean and uncluttered

### **Voiceover Options:**
- **Option A:** Professional voiceover (hire on Fiverr/Upwork)
- **Option B:** Founder's voice (authentic, personal)
- **Option C:** Text-only with captions (works well for international audience)

### **Music:**
- Upbeat, modern, tech-forward
- Not too distracting
- Fade in/out smoothly
- Consider: Epidemic Sound, Artlist, or royalty-free options

### **Editing Style:**
- Fast cuts (2-4 seconds per shot)
- Smooth transitions
- Zoom in on key actions
- Highlight important UI elements
- Keep it dynamic but not jarring

---

## KEY MESSAGES TO EMPHASIZE

1. **Problem → Solution:** Always show the transformation
2. **Speed:** Show how fast it works (AI generation, tree building)
3. **Organization:** Visual clarity of the knowledge trees
4. **Completeness:** Everything in one place (files, code, videos, docs)
5. **Reproducibility:** The research crisis → reproducible research
